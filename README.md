# onlypepes

monorepo using turbo 

- apps/web : onlypepes website
- packages/database: prisma db and onlypepes db functions
- packages/tsconfig: tsconfig for different packages

## dev

```bash
# docker service must be up
npm run db:up
npm run dev
```

## tests

```bash
# docker service must be up
npm run tests
```
